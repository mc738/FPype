## Import

### Import file

* Name: `import-file`
* Description: Import a file.

#### Properties

* `path` - The path to the file to be imported.
* `name` - The name of the data source to import to.

* Example:

```json
{
  "path": "/path/to/file",
  "name": "name"
}
```
